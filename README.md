# 研究助手 Agent

## 项目背景

在日常工作中，我经常需要：搜索某个话题的最新信息、整理分析、
然后生成报告。这个流程通常需要打开浏览器、复制粘贴、手动整理
——重复且低效。

研究助手 Agent 的目标是：用户只需用自然语言描述需求，
Agent 自动完成搜索、分析、保存的全流程。

## 功能

### 本次实现（MVP）
用户使用自然语言描述一个研究任务，Agent 自动完成以下操作：

- [x] 自动搜索网页获取最新信息
- [ ] 读取本地文件作为分析素材
- [ ] 执行 Python 代码进行数据处理
- [ ] 将分析结果保存到文件

### 未来迭代
- [ ] 支持 PDF 文件读取
- [ ] 支持多轮对话追问
